GitHub cheat sheet

Grundlagen Vertiefung

Neues Repository erstellen

  1. In GitHub neues Repository erstellen und benennen
  2. Ordner in VS Code ziehen und Terminal öffnen (Cmd+J)
  3. Folgende Befehle im Terminal eingeben:
    • git init [U = untract]
    • git add . [A = added]
    • git commit -m 'first commit'
    • git branch -m main
    • git remote add origin https://github.com/Melina412/_____.git (url des repo auf github)
    • git push -u origin main

Veränderungen in GitHub speichern

  1. Änderung im Dokument in VS Code [M = modified]
  2. Folgende Befehle im Terminal eingeben:
    • git add . [A = added]
    • git commit -m 'commit'
    • git push

Projekt klonen

  1. Lokalen Ordner erstellen
  2. Ordner in VS Code ziehen und Terminal öffnen (Cmd+J)
  3. Befehl eingeben: git clone https://github.com/Melina412/_____.git (github-url des repo was man klonen will)
